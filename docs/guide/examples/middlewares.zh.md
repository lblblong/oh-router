# 路由中间件

以下示例演示了如何使用路由中间件

## 在 React 中使用

```tsx
/**
 * inline: true
 */
import React from 'react'

export default () => (
  <iframe
    width="100%"
    height="500px"
    src="https://stackblitz.com/edit/oh-router-react-middlewares?embed=1&file=src/router/routes.tsx"
  ></iframe>
)
```
