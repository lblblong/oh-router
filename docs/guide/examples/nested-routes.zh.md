# 嵌套路由

以下示例演示了如何使用嵌套路由

## 在 React 中使用

```tsx
/**
 * inline: true
 */
import React from 'react'

export default () => (
  <iframe
    width="100%"
    height="500px"
    src="https://stackblitz.com/edit/oh-router-react-nested-routes?embed=1&file=src/router.tsx"
  ></iframe>
)
```

## 在 Vue 中使用

```tsx
/**
 * inline: true
 */
import React from 'react'

export default () => (
  <iframe
    width="100%"
    height="500px"
    src="https://stackblitz.com/edit/oh-router-vue-nested-routes?embed=1&file=src/router.ts"
  ></iframe>
)
```
