# 索引路由

以下是索引路由的基本案例

## 在 React 中使用

```tsx
/**
 * inline: true
 */
import React from 'react'

export default () => (
  <iframe
    width="100%"
    height="500px"
    src="https://stackblitz.com/edit/oh-router-react-index-route?embed=1&file=src/router.tsx"
  ></iframe>
)
```
